English:
This project analyzes data from Ice, a global online video game store, to identify the key factors that determine a game’s success. 
Using data on user and critic reviews, genres, platforms, ratings, and historical sales, the analysis seeks to uncover patterns that can 
guide marketing strategies and predict future performance.
The study focuses on data cleaning, preprocessing, and exploratory analysis, transforming raw data into actionable insights about sales trends, 
platform popularity, and audience preferences.

Español:
Este proyecto analiza datos de Ice, una tienda en línea de videojuegos con alcance global, para identificar los factores clave que determinan el éxito de un juego. 
Utilizando información sobre reseñas de usuarios y críticos, géneros, plataformas, clasificaciones y ventas históricas, 
el análisis busca descubrir patrones que ayuden a guiar estrategias de marketing y predecir el rendimiento futuro.
El estudio se centra en la limpieza, preprocesamiento y análisis exploratorio de datos, transformando datos sin procesar en información útil 
sobre tendencias de ventas, popularidad de plataformas y preferencias del público.
